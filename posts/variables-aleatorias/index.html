Variables aleatorias

By nsm

July 14, 2019

Existen ocasiones en las que de los fenómenos aleatorios, nos interesan ciertos valores numéricos asiciados a ellos. Por ejemplo, en el juego del truco un jugador puede, antes de ver sus cartas en una mano, preguntarse ¿a partir de qué número me conviene cantar envido (asumuendo en este caso que el contrincante no tema arriesgarse al punto de no querer aunque le convenga)? Es decir, del experimento aleatorio, que consiste en mezclar las cartas y repartirlas, al jugador en cuestión le interesa un número asociado al mismo, que en este caso es la suma de las dos cartas del mismo palo mayores (pero menores a 10).

Tenemos acá un espacio de probabilidad cuyo espacio muestral son todos los subconjunto de tamaño tres de un mazo de cartas españolas, cada uno de ellos equiprobable. Y el envido puede valer 0 o entre 20 y 33. Es decir que podemos decir que existe una función cuya imagen son estos números y cuya preimagen son ciertos subconjuntos del espacio muestral. Mas formalmente:

DEF Dado un espacio de probabilidad \((\Omega, \mathcal{F}, \mathcal{P})\), una variable aleatoria es una función \(X : \Omega \to \mathbb{R}\) tal que para todo \(x \in \mathbb{R}\) \[ \{\omega \in \Omega : X(\omega) \leq x\} \in \mathcal{F} \]

También podemos escribir también \((X < x)\) en lugar de \(\{\omega \in \Omega : X(\omega) \leq x\} \).

Podemos notar que para que el envido sea una variable aleatoria, el conjunto \(\mathcal{F}\) debe estar definido de una manera particular, de modo que induce una partición de \(\Omega\).

DEF Dada \(X\) una variable aleatoria definida sobre el espacio de probabilidad \((\Omega, \mathcal{F}, \mathcal{P})\), se llama función de distribución a la función \(F_X : \mathbb{R} \to [0,1]\) dada por: \[ F_X(x) = P(X \leq x) \]

Dicho de otra forma, el valor de la función de distribución de una variable aleatoria para un número real dado es la probabilidad de que dicha variable tome valores menores o iguales al mismo.

Observemos que \[ P(a < X \leq b) = P(X \leq b) - P(X \leq a) = F_X(b) - F_X(a) \]

Además, si \(g : \mathbb{R} \to \mathbb{R}\) es contínua y \(X\) es una variable aleatoria, entonces \(g(X)\) también es una variable aleatoria.

Veamos 3 ejemplos. Sean \(X\) una variable aleatoria, \(a, b \in \mathbb{R}\).

Ejemplo 1 Sea \(Y = aX + b\). Entonces

\( F_Y(x) = F_X(\frac{x-b}{a})\) si \(a>0\)

\( F_Y(x) = 1 - F_X(\frac{x-b}{a})\) si \(a<0\)

Esto porque si \(a > 0\), \[ F_Y(x) = P(Y \leq x) = P(aX+b \leq x) = P\left(X \leq \frac{x - b}{a}\right) = F_X\left(\frac{x - b}{a}\right) \]

Y si \(a < 0\), \[ F_Y(x) = P(aX+b \leq x) = P\left(X > \frac{x - b}{a}\right) = 1 - P\left(X \leq \frac{x - b}{a}\right) = 1 - F_X\left(\frac{x - b}{a}\right) \]

Ejemplo 2 Sea \(Y = X^2\), luego:

\[ F_Y(x) = \begin{cases} 0, & \text{para } x < 0\\ F_X(\sqrt{x}) - F_X(-\sqrt{x}), & \text{para } x \geq 0 \end{cases} \]

Si \(x \geq 0\):

\[ F_Y(x) = P(X^2 \leq x) = P(|X|<\sqrt(x)) = P(-\sqrt{x} < X < \sqrt{x}) = F_X(x) - F_X(-x) \]

Ejemplo 3 Sea \(Y = |X|\), luego:

\[ F_Y(x) = \begin{cases} 0, & \text{para } x \leq 0\\ F_X(x) - F_X(-x), & \text{para } x > 0 \end{cases} \]
